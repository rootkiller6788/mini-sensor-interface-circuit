# Course Dependency Tree

## Prerequisites
- mini-signal-system-theory: Fourier/Laplace, Shannon-Hartley
- mini-circuit-analysis: RC circuits, impedance
- mini-analog-electronics: Op-amp, CMRR
- mini-digital-electronics: Logic levels, LFSR/CRC
- mini-electromagnetic-wave: Paschen law, coupled lines
- mini-communication-principle: OOK modulation, Shannon

## Downstream
- mini-sensor-measurement: isolated ADC, iso-amp
- mini-industrial-fieldbus: isolated RS-485
- mini-emc-signal-integrity: CMTI, jitter, eye diagrams
- mini-electronic-mfg-test: test patterns, Weibull, BIST
